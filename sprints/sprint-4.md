TASK 1 — Extend src/storage/file_manager.py:
Add these export methods:

save_txt(segments, filepath):
  segments = list of {timestamp, text, translation}
  Format: "[00:01:23] text\n"

save_docx(segments, filepath, title="Recording Transcript"):
  Use python-docx library
  Header: title + recording date
  Table with 3 columns: Time | Transcript | Translation
  Column widths: 1.5cm | 8cm | 8cm
  Alternating row colors: white / #f5f5f5
  Bold header row with blue background #2196F3
  Footer: "Generated by RealTime Recorder"

save_srt(segments, filepath):
  Standard SRT subtitle format:
  1
  00:00:01,000 --> 00:00:05,000
  transcript text
  [translation text]
  
  Each segment = one subtitle entry
  Duration per segment: 4 seconds or until next segment

save_mp3(wav_filepath, mp3_filepath):
  Use pydub + ffmpeg
  If ffmpeg not found: show error "Install ffmpeg to export MP3"
  Bitrate: 128k
