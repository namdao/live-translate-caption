TASK 1 — Extend src/storage/file_manager.py: ✅

export_wav(audio_data, sample_rate, filepath):
  Writes in-memory numpy float32 array to WAV (PCM_16 subtype)
  ⚠️ CHANGED: was save_wav() reading from file — now writes from memory

export_transcript_txt(segments, filepath):
  Format: "[00:01:23] text\n" per segment (transcript only)
  ⚠️ CHANGED: was single export_txt() — split into transcript + translation

export_translation_txt(segments, filepath):
  Format: "[00:01:23] translation\n" per segment (translation only)
  ⚠️ NEW: separate file for translation text

export_docx(segments, filepath, title="Recording Transcript"):
  python-docx library
  Header: title + recording date
  Table: Time | Transcript | Translation (1.5cm | 8cm | 8cm)
  Alternating row colors: white / #F5F5F5
  Bold header row with blue background #2196F3
  Footer: "Generated by RealTime Recorder"

export_srt(segments, filepath):
  Standard SRT format
  Duration per segment: 4 seconds or until next segment start

export_mp3_from_data(audio_data, sample_rate, mp3_filepath):
  Converts in-memory numpy array → pydub AudioSegment → MP3 at 128k
  ⚠️ CHANGED: was export_mp3(wav_filepath, ...) reading from disk file
  If ffmpeg not found: raise RuntimeError("Install ffmpeg: brew install ffmpeg")
